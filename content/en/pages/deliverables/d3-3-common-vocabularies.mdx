This document discusses the task 3.1 (T3.1) of the InTaVia project ("Create a common in/tangible data model/ontology"), which aims to harmonize the controlled vocabularies used in the project's source datasets in order to integrate the heterogeneous datasets into a unified knowledge base. Some aspects have been discussed in other deliverables (see e.g. D3.1).

This deliverable is **public**.
